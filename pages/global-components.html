Cursor-follow (default)

Tooltip tracks the cursor; direction adds an offset in the named side. Hover any tile.

@[dir.label]

Locked to element

{ lock: true } pins the tooltip to the element edge; cursor movement no longer affects position.

@[dir.label]

Content shapes

The Tooltip surface renders whatever lives in its <slot>. The shape of $.tooltip.props is the host's contract. Here the same tooltip/Display.html picks between Equipment, Ability and Text variants based on prop shape.

@[ex.label]

Programmatic dismissal

Tooltips usually auto-hide on mouseleave, but you can dismiss explicitly (e.g. when starting a drag) via window.hideTooltip().

Trigger by type

notify(payload) accepts { error | warning | success | info: '...' } — the matching key sets the card's icon, color, and title. Auto-dismisses after 3 seconds.
